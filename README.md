# ICOS (Integral Commons OS)
A decentralized, federated governance and coordination system for human and collective flourishing.
## Core Principles
- Stewardship over ownership
- Revocability of power
- Structured deliberation before decisions
- Local-first and federated architecture
- Privacy and user control
## System Overview
ICOS is composed of:
- Governance Engine
- Deliberation System
- Commons Knowledge Layer
- Federated Identity
- AI-assisted sensemaking
