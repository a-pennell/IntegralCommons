# CommonGround UX Flows

## Overview
UX is critical to governance quality. The system prioritizes structured deliberation over chaotic discussion.

---

## Proposal Creation Flow

1. User clicks "Propose Change"
2. Define scope
3. System suggests:
   - Electorate
   - Thresholds
4. Draft proposal
5. Submit for support

---

## Deliberation Flow

Structured layout:

- Perspectives
- Arguments
- Evidence
- Synthesis (AI-assisted)

### Features
- "Steelman" opposing view
- "What am I missing?"
- Disagreement mapping

---

## Voting Flow

- Clear summary
- Key arguments
- Risks
- Participation metrics
- Quorum tracking

---

## Delegation Dashboard

- Who holds authority
- Scope of power
- Revocation status
- Performance indicators

---

## Conflict Resolution Flow

1. Mediation attempt
2. Structured dialogue
3. Escalation if needed
4. Transparent decision process
