# ICOS Overview

Integral Commons OS (ICOS) is a decentralized infrastructure for:
- Decision-making
- Governance
- Knowledge coordination
- Collective intelligence

It integrates:
- Commons governance (Ostrom)
- Integral theory (Wilber)
- Process philosophy (Whitehead)
- Developmental psychology
- Distributed systems design

---

## Core Layers

1. Personal (self-awareness)
2. Relational (group coordination)
3. Collective (governance)
4. Ecological (stewardship)
5. AI (sensemaking)

---

## Design Goal

Enable systems that are:
- Adaptive
- Participatory
- Resistant to capture
- Developmentally supportive
